# Nightscape Stacker — Releases

This repository hosts the public release artifacts for [Nightscape Stacker](https://nightscapestacker.com), a desktop application for stacking and processing nightscape astrophotography images.

The source code is maintained in a private repository. Releases are published here automatically by CI on every build.

## About Nightscape Stacker

Nightscape Stacker is a desktop tool built for landscape astrophotographers. It combines multiple exposures into a single clean image — reducing noise, preserving foreground sharpness, and removing gradients and stars — without requiring any command-line tools or technical setup.

Learn more at [nightscapestacker.com](https://nightscapestacker.com).
